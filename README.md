##Live Site
https://rjcolbert.github.io/my_site/

##Overview
This is a hand-coded rebuild of my Framer About page using HTML, CSS, and JavaScript.

##Focus
Create a hand-coded site to use in a discussion to describe who I am, what my interests are, and how I think and connect with my work.

##Approach
The structure is kept intentionally simple:
- Semantic HTML for content structure
- CSS for layout, responsiveness, and animation
- JavaScript for interaction behavior and content structure

##Notes
Site is responsive for the most part, but there are some SVG issues in the Safari browser.
